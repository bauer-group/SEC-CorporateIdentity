## Logo-Farbspezifikation (Print)

Verbindliche Farbwerte für alle Druckmaterialien:

| Farbe | CMYK / Sonderfarbe | RGB (Screen) | Hinweis |
|-------|-------------------|--------------|---------|
| **Orange** | PMS 144 C/U | `#FF8500` | Primärfarbe, Bildmarke |
| **Weiß** | Volltonfarbe Weiß | `#FFFFFF` | RAL 9016 (Verkehrsweiß) |
| **Schwarz** | Volltonfarbe Black | `#000000` | Wortmarke "BAUER" |
| **Grau** | PMS 419 C/U | `#2F3234` | RAL 7021 (Schwarzgrau), Wortmarke "GROUP" |

::: warning Logo-Rahmen
Der Rahmen um das Logo soll schmäler gehalten werden — ca. ½ bis ¼ der Standardbreite (Haarlinie). Details siehe Druckvorlagen.
:::

## Telefonnummern-Format

Vorwahlen werden **ohne Klammern** dargestellt. Gilt einheitlich für alle Geschäftsdrucksachen.

| Format | Beispiel |
|--------|----------|
| **Telefon** | Tel. +49 9971 20098 - 100 |
| **Fax** | Fax +49 9971 20098 - 4100 |
| **Mobil** | Mobil +49 178 1444855 |

## Allgemeine Druckhinweise

Alle Druckmaterialien folgen den Spezifikationen aus [Print & Produktion](/de/print/):

- Farben: CMYK-Werte verwenden, kein RGB
- Sonderfarbe: Pantone PMS 144 C (wo möglich)
- Papier: Siehe [Papierempfehlungen](/de/print/papier)
- QR-Codes: Immer über `go.bauer-group.com` — siehe [QR-Code-Regeln](/de/werbemittel/bloecke#qr-code-regeln-alle-printmaterialien)
