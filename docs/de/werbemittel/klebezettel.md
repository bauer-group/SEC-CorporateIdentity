# Klebezettel (Haftnotizen)

Quadratische Haftnotizen im BAUER GROUP Design. Das Layout folgt dem gleichen System wie die [Schreibblöcke](/de/werbemittel/bloecke) — angepasst auf das kleinere Format.

## Spezifikationen

| Eigenschaft | Wert |
|-------------|------|
| **Format** | Quadrat, 72 × 72 mm |
| **Papier** | 80 g/m² Offsetpapier |
| **Blatt** | 50 Blatt |
| **Bindung** | Leimung am Kopf |
| **Bedruckung** | Vierfarbig (4/0) |
| **Datencheck** | Ohne |
| **Bestellmenge** | 1.000 Stück |
| **Version** | 2023-06-26 |

## Layout-Aufbau

| Element | Position | Details |
|---------|----------|---------|
| **Logo** | Oben links | Wide-Variante mit Bildmarke, klein skaliert |
| **QR-Code + C2A** | Oben rechts | Info-Icon (ℹ-Sprechblase + Pfeil in Orange) + QR-Code |
| **Wasserzeichen** | Zentriert | "B"-Bildmarke, groß, ~5–10% Deckkraft |
| **Website** | Unten rechts | `www.bauer-group.com` |
| **Akzentbalken** | Unterkante | BAUER Orange (#FF8500) mit grauem Abschluss rechts |

::: tip Gestaltungsprinzip
Die Klebezettel nutzen exakt das gleiche Gestaltungssystem wie die Schreibblöcke — Logo oben links, QR+C2A oben rechts, Wasserzeichen zentral, Akzentbalken unten. Kein Raster und kein Lineal aufgrund des kleineren Formats.
:::
