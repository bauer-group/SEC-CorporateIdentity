# Mappen

## Spezifikationen

| Eigenschaft | DIN-Lang | DIN-A4 |
|-------------|----------|--------|
| **Format** | DIN-Lang, Querformat (#51) | DIN-A4, Hochformat |
| **Ausstattung** | Verschlusslasche | Visitenkartenschlitz |
| **Papier** | 300 g/m² Bilderdruck, matt gestrichen | 300 g/m² Bilderdruck, matt gestrichen |
| **Veredelung** | Glänzende Cello, einseitig | Glänzende Cello, einseitig |
| **Bedruckung** | Vierfarbig (4/0), Rückseite unbedruckt | Beidseitig vierfarbig (4/4) |
| **Farbprofil** | ISO 12647-2 (PSO) | ISO 12647-2 (PSO) |
| **Datencheck** | Ja | Ja |
| **Bestellmenge** | 1.000 Stück | 1.000 Stück |
| **Version** | 2023-07-14 | 2023-06-26 |

## Gestaltung

### Gemeinsame Elemente

| Element | Details |
|---------|---------|
| **Fotomotiv** | Großflächiges Bildmotiv (Büro-/Team-Szenen) als Hintergrund |
| **Logo** | BAUER GROUP Logo (Weiß auf Schwarz) im unteren Bereich |
| **Website** | `www.bauer-group.com` (Weiß auf schwarzem Balken) |
| **QR-Code** | Produktspezifischer QR-Code |
| **Orange-Akzent** | Orangefarbener vertikaler Balken (DIN-Lang) bzw. Taschenklappe (A4) |
| **Grau-Akzent** | Grauer Flächenakzent als Gegenpol |

### QR-Codes

| Produkt | QR-Code URL | Shortlink-ID |
|---------|-------------|--------------|
| **Mappe DIN-Lang** | `https://go.bauer-group.com/p-folderdl` | `p-folderdl` |
| **Mappe DIN-A4** | `https://go.bauer-group.com/p-foldera4` | `p-foldera4` |
