# Visitenkarten

Doppelseitige Visitenkarten der BAUER GROUP — Vorderseite in der primären Sprache (Deutsch oder Chinesisch), Rückseite in Englisch. Jede Karte enthält einen personalisierten QR-Code zum LinkStack-Profil des Mitarbeiters.

## Sprachvarianten

| Variante | Vorderseite | Rückseite |
|----------|-------------|-----------|
| **DE / EN** | Deutsch | Englisch |
| **CN / EN** | Chinesisch | Englisch |

## Layout — Vorderseite (Primärsprache)

| Element | Position | Details |
|---------|----------|---------|
| **Logo** | Oben rechts | Wide-Variante mit Bildmarke |
| **Firmenname** | Oben links | "BAUER GROUP GmbH & Co. KG", fett |
| **Adresse** | Unter Firmenname | Janahof 30 · D-93413 Cham (Deutschland) |
| **Kontaktdaten** | Links unten | Tel., Fax, Mobil, E-Mail, Website (Vorwahlen ohne Klammern) |
| **Name + Titel** | Rechts unten | Name (fett), Position darunter |
| **Wasserzeichen** | Links, zentriert | "B"-Bildmarke, dezent (~10% Deckkraft) |
| **Orange Aufzählungspunkte** | Vor Firmenname/Adresse | Kleine orange Quadrate als Listensymbole |
| **QR-Code** | Geplant | Personalisierter Link zum LinkStack-Profil |

### Telefonnummern-Format

## Layout — Rückseite (Englisch)

Gleiches Layout wie Vorderseite, alle Texte in Englisch (Germany, Phone, Managing Director etc.).

## QR-Code & LinkStack

Jede Visitenkarte enthält einen **personalisierten QR-Code**, der zum individuellen LinkStack-Profil des Mitarbeiters führt.

| Eigenschaft | Wert |
|-------------|------|
| **Plattform** | [LinkStack](https://linkstack.org/) (selbst gehostet) |
| **URL-Schema** | `links.bauer-group.com/{vorname.nachname}` |
| **Beispiel** | `links.bauer-group.com/karl.bauer` |
| **Platzierung** | Vorderseite (Primärsprache) |

::: tip Personalisierung
Der QR-Code ist pro Mitarbeiter individuell. Bei Bestellung neuer Visitenkarten muss der Name im URL-Schema angegeben werden. Das LinkStack-Profil wird zentral verwaltet und kann jederzeit aktualisiert werden — ohne Neudruck der Karten.
:::
